처음 연동하시나요? 3분 설정 가이드 보기
  1. 구글 시트 만들기 — 구글 드라이브에서 새 스프레드시트를 만들고 URL의 /d/와 /edit 사이 값을 "스프레드시트 ID"에 붙여넣습니다.
  2. 구글 클라우드 프로젝트 생성 — console.cloud.google.com 접속 → 새 프로젝트 생성.
  3. API 사용 설정 — "API 및 서비스 → 라이브러리"에서 Google Sheets API를 사용 설정합니다.
  4. OAuth 동의화면 구성 — 사용자 유형 "외부", 테스트 사용자에 본인 계정 추가.
  5. 사용자 인증정보 생성 — "OAuth 클라이언트 ID → 웹 애플리케이션", "승인된 자바스크립트 원본"에 이 파일을 열 주소를 등록 (file://는 불가).
  6. 발급된 클라이언트 ID를 입력하고 "구글 계정으로 로그인"을 누르면 완료.
