📲 텔레그램 실시간 알림 (회원가입·결제·배송지 자동 수신)

아래에 봇 토큰과 Chat ID를 입력해 저장하면, 신규 회원가입과 주문 결제 완료(결제내용·배송지 포함)가 발생할 때마다 관리자 텔레그램으로 실제 메시지가 전송됩니다.

텔레그램 알림 상태: 미설정

❓ 봇 토큰 / Chat ID 받는 방법 (처음 1회만)

1단계 · 봇 만들기 (공통)

  1. 텔레그램에서 @BotFather 검색 → /newbot → 봇 이름 정하면 봇 토큰을 줍니다. 위 칸에 붙여넣기.

2단계-A · 👥 그룹방으로 받기 (여러 명이 함께 볼 때)

  1. 알림 받을 그룹방에 내 봇을 초대합니다 (그룹 → 멤버 추가 → 봇 이름 검색).
  2. 그 그룹방에서 아무 메시지나 보냅니다. (예: /start)
  3. 위 [🔍 봇이 속한 채팅방(그룹) ID 자동 찾기] 클릭 → 목록에서 그룹을 선택하면 Chat ID가 자동 입력됩니다.
  4. [알림 설정 저장] → [🔔 테스트 전송] 으로 그룹에 오는지 확인!

※ 그룹 Chat ID는 마이너스(-)로 시작합니다 (예: -1001234567890). 개인 ID(양수)를 넣으면 그룹엔 안 옵니다.

2단계-B · 👤 개인 채팅으로 받기

  1. 내 봇을 검색해 대화창에서 아무 메시지나 보냅니다.
  2. 위 [🔍 자동 찾기]를 누르거나, @userinfobot 으로 내 Chat ID(숫자)를 확인해 입력.

❗ 그룹에 안 온다면: ①봇이 그룹에 초대됐는지 ②그룹에서 메시지를 한 번 보냈는지 ③Chat ID가 -로 시작하는 그룹 ID인지 확인하세요.

🔔 (참고) 카카오톡 알림톡 채널명
